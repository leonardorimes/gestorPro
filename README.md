# GestorPro

## Descrição

Sistema web para gestão de clientes e serviços, desenvolvido com foco em boas práticas de frontend e backend, simulando um ambiente real de produto.

## Stack

- TypeScript
- React
- Next.js
- Node.js
- PostgreSQL
- Git

## Funcionalidades

- Autenticação de usuários
- Gestão de clientes
- Gestão de serviços
- Ordens de serviço
- Dashboard com indicadores básicos

## Como rodar o projeto

1. Clone o repositório
2. Instale as dependências
   ```bash
   npm install
   ```
